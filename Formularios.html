Historia de la Web

Este lenguaje de programación Web fue inventado por Tim Berners-Lee en 1986.
El HTML combina el concepto de Hipertexto, que permite conectar dos elementos entre si, y el SGML (Lenguaje Estándar de Marcación General) que permite colocar etiquetas o marcas en un texto para indicar como queremos que ese documento se visualice.
